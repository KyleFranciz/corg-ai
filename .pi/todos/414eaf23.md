Create a transcription component that loads on session pages so users can continue asking additional questions tied to that same session context.

## Goals
- Add a reusable transcription UI component for session pages.
- Load and display session-specific transcription/history context.
- Enable users to submit follow-up questions within the active session.

## Acceptance criteria
- Transcription component renders on session detail/page views.
- Component is bound to the current session id/context.
- User can ask additional questions and receive responses in-session.
- Session continuity is preserved across follow-up turns.
- Empty/loading/error states are handled clearly.
