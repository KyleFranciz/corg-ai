Ensure each user/session has its own persisted context history so the agent can reference prior messages/actions within that session. This should improve context awareness and response continuity.

Acceptance ideas:
- Context is scoped by session ID (no cross-session leakage)
- History persists locally/offline-first
- Agent retrieval uses only current session history by default
- Clear lifecycle handling (create/load/update/cleanup)
