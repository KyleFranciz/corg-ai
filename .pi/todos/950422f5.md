Validate that deleting a session fully removes all associated local artifacts.

Checklist:
- Create a test session with representative related data (messages, files, embeddings/vector entries).
- Delete the session via UI flow.
- Verify backend/local stores no longer contain session-linked records.
- Verify vector store no longer has entries for deleted session.
- Verify no broken references remain in UI/state after app reload.
- Verify behavior for invalid/nonexistent session ID.

Acceptance criteria:
- Deletion is complete and consistent across app layers.
- No orphaned records/files/vector entries remain.
- UX and API error handling behave as expected.
