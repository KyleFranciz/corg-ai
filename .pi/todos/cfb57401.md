Add a markdown library on the frontend to render LLM responses with proper formatting (headings, lists, code blocks, links, emphasis, etc.).

## Goals
- Integrate a markdown parser/renderer into the frontend response view.
- Ensure LLM text is displayed with correct markdown formatting.
- Keep rendering safe and predictable for user-facing output.

## Acceptance criteria
- LLM responses supporting markdown render correctly in the chat UI.
- Code blocks, lists, links, and emphasis are displayed properly.
- Rendering path is integrated where assistant responses are shown.
- Any needed sanitization/safe rendering safeguards are in place.
